# Syncfusion® Vue UI Components Skills

**Agent Skills** for building accurate, production-ready applications with Syncfusion Vue components. Install these skills into your IDE (VS Code, Cursor, CodeStudio) to enable AI assistants to generate correct Syncfusion Vue code grounded in official APIs, patterns, and best practices.

## Key Features

- **Comprehensive Component Coverage** — Skills for Syncfusion Vue UI components with detailed implementation patterns
- **Production-Ready Code** — Official APIs, props, events, and methods with proven best practices
- **Multi-Purpose Skills** — Both common/utility skills and component-specific skills for flexible AI-assisted development
- **Built-in Theme Knowledge** — Syncfusion theme configurations and styling capabilities
- **Best Practices** — State management patterns, feature integration, and globalization (RTL, i18n/l10n)
- **Rich References** — Implementation guides and code examples for common use cases and advanced features

## Quick Start

### Install Skills

```bash
npx skills add syncfusion/vue-ui-components-skills -y
```

Your AI assistant will automatically load these skills in supported IDEs.

### Explore Skills

Each skill is self-contained:
- **SKILL.md** — Core concepts and patterns
- **references/** — Detailed implementation guides by feature area

## How AI Assistants Use These Skills

With skills installed, your AI assistant:
1. **Loads** relevant skills based on your query (e.g., "How to implement scheduler with timeline week view?" → `syncfusion-vue-scheduler`)
2. **Extracts** accurate APIs, props, and proven patterns
3. **Generates** production-ready code with proper imports and types
4. **Enforces** best practices (theming, accessibility, performance)
